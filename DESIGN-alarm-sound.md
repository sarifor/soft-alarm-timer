# 설계 문서: 알람 사운드 추가

## 개요

타이머 완료 시 부드러운 벨소리를 자동 재생한다.

## 결정 사항

| 항목 | 결정 | 이유 |
|------|------|------|
| 사운드 파일 방식 | Web Audio API로 코드 생성 | 외부 파일 불필요. 사인파 + 감쇠로 부드러운 벨소리 구현 |
| 파일 위치 | 없음 | 파일 번들 불필요 |
| 재생 타이밍 | `finish.html` 로드 시 자동 재생 | 타이머 완료 = 팝업 열림 |
| 재생 횟수 | 1회 | 소프트한 벨소리라 반복 불필요 |
| 볼륨 | 1.0 (기본값) | |
| 재생 중단 | Dismiss·Restart 클릭 시 `audio.pause()` | |

## 변경 파일

| 파일 | 변경 내용 |
|------|----------|
| `finish.js` | Web Audio API 오디오 재생 로직 추가 |

## 변경 없는 항목

- `manifest.json` — 추가 권한 불필요
- 오디오 파일 없음
